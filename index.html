Bluetooth GPS/GNSS connector on Android for OpenStreetMap Mappers

Description

You can use an external GPS/GNSS reciever as same as internal one. Connect external Bluetooth Global Navigation Satellite System (GNSS) receivers, using the GPS, GLONASS, Galileo, Beidou or QZSS system, to your Android device.

Screenshots

Rador Interface: Each satellites have prefix in name;

  • G: GPS
  • R: GLONASS
  • E: Galileo
  • S: SBS(WAAS, MSAS, EGNOS, etc)
  • B: Beidou/Compass
  • Q: QZSS
